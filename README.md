# EduManage - Advanced Student Data Table

A modern, production-ready data table built with **Next.js 15**, **TypeScript**, **TanStack Table**, and **ShadCN UI**.

## ✨ Key Features

- **Fully Responsive** - Seamless table & card layout on all devices
- **Global Search** with debounce for fast filtering
- **Multi-Column Filtering** (Status, Priority, Course, etc.)
- **Advanced Sorting** (Ascending / Descending) on all columns
- **Pagination** with customizable page sizes
- **Row Selection** + **Bulk Actions** (Delete, Export)
- **Column Visibility Toggle** - Show/hide any column
- **Dark & Light Mode** support
- **File Upload** with Drag & Drop + validation (in form)
- **CSV Export** functionality
- **Real-time CRUD** operations (Mock API ready)
- **Loading, Empty & Error States** handled gracefully
- **Optimized Performance** for 500+ records

### Beautiful UI Elements
- Gradient avatars with student initials
- Progress bars with percentage
- Status & Priority colored badges
- Hover effects and smooth transitions
- Professional action menus

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Table**: TanStack React Table v8
- **UI**: ShadCN UI + Tailwind CSS
- **Icons**: Lucide React
- **Date Handling**: date-fns
